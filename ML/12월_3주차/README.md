환경 먼저 세팅을 해준다

Cart-Pole의 상태는 4차원 벡터(배열)로 각 값은 다음과 같은 의미를 갖는다는 것을 잊지 말자
환경 변수
  1. x : Cart의 가로상의 위치
  2. θ : Pole의 각도
  3. dx/dt : Cart의 속도
  4. dθ/dt : θ의 각속도

게임이 끝나는 상황:

1. θ가 15˚이상이 되었을 떄,
2. 원점으로부터의 x의 거리가 2.4이상이 되었을 떄.

에이전트가 취할 수 있는 행동은 다음 두가지 이다

1. left 이동
2. right 이동
